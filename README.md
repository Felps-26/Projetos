## 🚀 Projetos Destacados

- **Hello World**: Primeiro programa em Java

## 💻 Como Usar

### Para projetos em C:
```bash
# Compilar
gcc arquivo.c -o programa

# Executar
./programa
```

### Para projetos em Java:
```bash
# Compilar
javac Arquivo.java

# Executar  
java Arquivo
```
